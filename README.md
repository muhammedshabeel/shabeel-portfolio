# Shabeel Final Production Portfolio

React + Vite + Tailwind portfolio with locked dependencies and premium UI.

## Run locally

```bash
cd shabeel-portfolio-final-production
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this folder to GitHub.
2. Import the repository into Vercel.
3. Use:
   - Framework: Vite
   - Install Command: npm install
   - Build Command: npm run build
   - Output Directory: dist

## Notes

Dependencies are locked to avoid npm conflicts:
- Vite 5
- @vitejs/plugin-react 4.2
- Tailwind 3.4.4
